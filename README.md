# JavaFX를 활용한 미니게임

## 개발 인원
- 팀 프로젝트(2인)
- 이유진(본인), <a href="https://github.com/SuYoungSong">송수영</a>

## 주요 기능
- **오목**
	1. 착수 시간제한
	2. 렌주룰 적용
- **끝말잇기**
	1. 표준 국어 대사전에 등록된 단어인지 확인
	2. 혼자하는 버전과 컴퓨터와 할 수 있는 버전으로 구분
- **2048**
	1. 최고점수
	2. 타일 합치기
- **부루마블**
	1. 글 쓰기, 수정
	2. 댓글
	3. 좋아요
